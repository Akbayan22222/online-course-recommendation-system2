# NKU SmartPath

Магистрлік диссертация контекстіне арналған Vite + React + Tailwind CSS веб-платформа прототипі.

## Фичалар

- Ikigai Onboarding: 3 сұрақ арқылы мақсатты анықтау
- EduBalancing: академиялық энергияға қарай ұсыныстарды теңгеру
- Интеллектуалды 3 карточка: мақсат, релакс, hard-skill
- Dark mode + glassmorphism + Framer Motion
- Қазақ/орыс i18n
- Recharts негізіндегі эксперименттік аналитика

## Іске қосу

```bash
npm install
npm run dev
```

Сосын браузерде Vite көрсеткен URL мекенжайын ашыңыз.
